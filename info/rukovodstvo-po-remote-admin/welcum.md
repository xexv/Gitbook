# WELCUM

## <mark style="color:yellow;">Добро пожаловать!</mark>

Представляем четвертую версию Remote Admin , официального инструмента управления сервером SCP: Secret Laboratory. Это позволяет тебе отправлять запросы серверу, используя удобный интерфейс.

## <mark style="color:yellow;">Как им пользоваться?</mark>

1. Выбери команду, которую хочешь выполнить.
2. Внимательно прочти описание, чтобы убедиться в его функции - иногда бывает трудно отменить команду.
3. Соответствующим образом отрегулируй параметры и, при необходимости, выбери игрока(ов), на которого должна быть нацелена команда. (<mark style="color:green;">СОВЕТ</mark>: Удерживай <mark style="color:red;">CTRL</mark>, чтобы выбрать сразу нескольких игроков.)
4. Подтверди запрос.

## <mark style="color:yellow;">Давайте начнем!</mark>

Теперь ты готов начать ~~издеваться~~ помогать игрокам. Развлекайся!

## <mark style="color:yellow;">Четвертая редакция</mark>

Эта редакция RA улучшает меню за счет появления QoL функций, а также дает возможность настраивать свой интерфейс; благодаря симпатичному значку шестеренки в правом верхнем углу этого окна.

<mark style="color:green;">СОВЕТ</mark>: при наведении курсора на некоторые пункты меню отобразится полезная информация!
