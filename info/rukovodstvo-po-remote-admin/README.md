# Руководство по Remote Admin

## Практика

На сервере <mark style="color:orange;">Luminus</mark> использовать RA в личных целях запрещено, так что вы можете попрактиковать свои навыки на выделенном локальном сервере. Шаги:

1. Зайти в игру
2. В главном меню нажать на `ё` или `~` , таким образом вы откроете консоль
3. Ввести в консоли `host` и нажать <mark style="color:red;">Enter</mark>
4. Выбрать вариант `все равно создать/create anyway`

Сервер создан, осталось выдать себе на нем права администратора:

1. Чтобы выдать себе права снова открываем консоль и вводим `adminme` и нажимаем <mark style="color:red;">Enter</mark>

{% hint style="info" %}
`adminme` не будет работать, если сервер полностью не запустился
{% endhint %}

## Основные категории

Открывается панель на русский `Ь`/ английскую `M` .

### 1. Секция PLAYERS

* Быстрое взаимодействие со списком игроков
  1. _<mark style="color:red;">CTRL</mark>_+_<mark style="color:red;">A</mark>_ - выделение всех игроков
  2. _<mark style="color:red;">SHIFT</mark>_+_<mark style="color:red;">LMB</mark>_ - выделение всех ниже твоего клика
  3. _<mark style="color:red;">CTRL</mark>_+_<mark style="color:red;">LMB</mark>_ - выборочное выделение
* Сортировка списка игроков
  1. Player ID’s - по id (1,2…)
  2. Name - по алфавиту
  3. Class - по ролям (МОГ, Ученые…)
  4. Team - по командам
