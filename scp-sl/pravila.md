# Правила

Данный документ предназначен для прочтения перед игрой на сервере <mark style="color:$info;background-color:orange;">Milkis</mark>. Мы за комфортную и приятную игру, так что соблюдение правил **строго обязательно**.

Сотрудники сервера в праве выдавать наказания по своему усмотрению, если ситуация является спорной и не указана в правилах. При несогласии с выданным наказанием, ты можешь подать на пересмотрение в наш [D**iscord**](https://discord.gg/invite/edzu4BS8J8) (жмяк). Рассмотрение занимает определенное время, потому что каждый сотрудник работает в свободное время и по собственной инициативе, так что прояви терпение, тебе обязательно ответят.

***

## Основное для всех серверов

{% hint style="info" %}
[Политика блокировок](https://scpslgame.com/Ban_Policy.pdf) всегда имеет приоритет над любыми правилами сервера <mark style="color:$info;background-color:orange;">Milkis</mark>.
{% endhint %}

* Если что-то не указано в правилах, это не означает, что такие действия разрешены.
* Незнание правил не освобождает от ответственности.
* Сотрудники могут попросить игрока прекратить делать что-либо, обсуждать или говорить о чем-то, если это что-то неподобающее или нежелательное.

***

### M.1 Стороннее ПО

Использование сторонних программ, предоставляющих преимущество перед другими игроками, строго запрещено.

Если у администрации возникают подозрения о применении стороннего ПО, она имеет право запросить проверку игрока через Discord.

{% hint style="danger" %}
50 лет.
{% endhint %}

***

### M.2 Багоюз

**Багоюз** — использование игровых багов/недоработок/уязвимостей для получения преимущества или нарушения игрового процесса — строго запрещён.

{% hint style="info" %}
Если вы случайно столкнулись с багом, сообщите об этом администрации сервера.
{% endhint %}

{% hint style="danger" %}
Предупреждение → до 1 дня
{% endhint %}

***

### M.3 Нарушение стабильности

Запрещено проведение действий, способных нарушить стабильную работу сервера и игрового процесса. Нарушения включают, но не ограничиваются:

{% hint style="warning" %}
* **Многократное нарушение правил**.
* **Злоупотребление эксплойтами и багами игры.**
* **DoS/DDoS-атаки** или любые попытки нарушения сетевой инфраструктуры.
* **Организованное нарушение правил**: скоординированные действия с целью помешать работе сервера или другим игрокам.
* **Подстрекательство** к нарушениям или деструктивным действиям.
* **Попытка нарушить работоспособность сервера** через сторонние программы, модификации или любые иные методы.
{% endhint %}

{% hint style="danger" %}
В зависимости от тяжести.

Простая нагрузка - до 3 часов.

Тяжелая нагрузка - 50 лет.
{% endhint %}

***

### M.4 Реклама

{% hint style="warning" %}
Запрещено рекламировать:

* Сторонние проекты игры SCP:SL.
* Постороннее ПО.
* Аккаунты социальных сетей (YouTube, Twitch и другие платформы).
* Игровые сайты (интернет-казино и сайты по ставкам).
{% endhint %}

{% hint style="success" %}
Игрокам, имеющим роль **“Контент-Мейкер”**, разрешено рекламировать свои социальные сети.
{% endhint %}

{% hint style="danger" %}
Предупреждение → 50 лет
{% endhint %}

***

### M.5 Голосовой чат

Запрещено транслировать громкие и/или неприятные звуки, которые могут мешать игровому процессу и комфортному общению игроков. Так же, запрещено включать музыку в микрофон находясь в спектаторах или через SCP-1576 (Коммуникатор загробной жизни)

{% hint style="warning" %}
Если кто-то попросил вас прекратить воспроизведение громких и/или неприятных звуков, вы обязаны выполнить просьбу. (не относится к спектаторам и коммуникатору)
{% endhint %}

{% hint style="danger" %}
Предупреждение → мут до 3 часов
{% endhint %}

***

### M.6 Обход блокировки

Запрещено использовать дополнительные аккаунты (твинки) для нарушения правил сервера или обхода бана/любого вида наказания.

{% hint style="info" %}
**Пример:** Если вы получили бан на одном аккаунте, все последующие аккаунты, используемые для обхода наказания, будут забанены на х2 **наказание основного аккаунта**, независимо от их уровня в Steam или других характеристик.
{% endhint %}

{% hint style="danger" %}
Соответствующие предыдущему х2
{% endhint %}

***

### M.7 Обман и помеха

Запрещено обманывать или мешать участникам/администрации сервера. Это включает в себя выдачу себя за администрацию, а также намеренное вмешательство в процесс разборок или работу администрацией.

{% hint style="danger" %}
Предупреждение → до 3 часов.
{% endhint %}

***

### M.8 Неадекватное поведение

Запрещены любые формы оскорблений, дискриминации или агрессии по отношению к другим игрокам, включая, но не ограничиваясь, оскорблениями по расовому, национальному, этническому, половому или религиозному признаку. Это включает, но не ограничивается:

{% hint style="warning" %}
* Оскорбления игроков на основе их расы, пола или национальности.
* Призыв к насилию или уничижительное поведение в отношении других игроков.
* Дискриминационные высказывания и действия.
{% endhint %}

Мы понимаем, что игроки могут развеселиться и начать делать/говорить не совсем хорошие,но смешные вещи. Однако, нужно помнить о том, что может быть весело для тебя, будет обидно для другого… так что, пожалуйста, знайте меру веселья.

{% hint style="danger" %}
Предупреждение -> мут до 1 часа -> бан до 3 часов
{% endhint %}

***

### М.9 Нарушение законодательства

Запрещены любые действия, нарушающие действующее законодательство РФ. Это включает, но не ограничивается:&#x20;

{% hint style="warning" %}
* распространение незаконного контента
* призывы к насилию
* экстремизм
* любые другие действия, противоречащие закону
{% endhint %}

{% hint style="danger" %}
Предупреждение → бан до 3 часов → бан до 3 дня.
{% endhint %}

***

### M.10 Доксинг и раскрытие личной информации

Запрещено собирать, распространять или использовать личную информацию других участников без их явного согласия. Это включает, но не ограничивается:

{% hint style="warning" %}
* Публикацию реальных имён, адресов, номеров телефонов, электронных почт или других контактных данных.
* Раскрытие информации о месте работы, учёбы или личной жизни.
* Распространение фотографий, переписок или любой другой конфиденциальной информации.
{% endhint %}

{% hint style="danger" %}
* Первое нарушение: мут до 3 часов.
* Повторное нарушение: бан на 24 часа.
* Серьёзные случаи (например, угрозы или шантаж): перманентный бан на проекте.
{% endhint %}

## Classic

Ниже приведены правила, которые относятся исключительно к классическому серверу.

{% hint style="info" %}
Также советую ознакомится с [взаимодействием классов](../info/vzaimodeistvie-klassov.md).
{% endhint %}

***

### C.1 Порча игрового процесса

Запрещено совершать действия, которые намеренно нарушают или ухудшают игровой процесс для других игроков.

{% hint style="warning" %}
1. Намеренное создание помех и руин союзным классам:
   * Нанесение урона по союзникам.
   * Закрытие дверей перед союзниками.
   * Порча или воровство предметов союзников.
   * Суицид, приводящий к ухудшению положения команды.
   * Развязывание врага или нейтрала в зоне побега.
2. Намеренное затягивание раунда (например, избегание выполнения игровых целей без видимой причины).
3. Блокировка важных объектов и мест (например, закрытие оружейных, терминалов и других зон с целью помех).
{% endhint %}

{% hint style="success" %}
Действия, вызванные угрозой жизни (например, побег от SCP).
{% endhint %}

{% hint style="danger" %}
Предупреждение -> до 6 часов
{% endhint %}

***

### C.2 Кооператив враждебных классов

Кооперация между враждебными классами запрещена. Это включает в себя сотрудничество, обмен ресурсами или совместные действия. (см. [vzaimodeistvie-klassov.md](../info/vzaimodeistvie-klassov.md "mention"))

{% hint style="warning" %}
* Обмен информацией между SCP и людьми через чат или другие каналы связи.
* Намеренное игнорирование врагов из противоположного класса (например, учёные, пропускающие охрану Хаоса).
* Совместное устранение или атака на другой враждебный класс (например, SCP и МТФ против Хаоса).
{% endhint %}

{% hint style="success" %}
Кооперация возможна, если одна из сторон находится в состоянии полной беззащитности (связанный или безоружный игрок).
{% endhint %}

{% hint style="danger" %}
Предупреждение -> до 3 часов
{% endhint %}

***

### C.3 Кемперство

Запрещено кемпить в какой-либо позиции затягивая раунд или в случае SCP кемпить игроков больше 2 минут.

{% hint style="success" %}
Можно кемпить точки, которые не являются единственным выходом. (Гейты, лифты, чекпоинты).

Скромник (SCP - 096) может кемпить, если он единственный SCP или с ним только Компьютер (SCP - 079).
{% endhint %}

{% hint style="danger" %}
Предупреждение -> Перевод в спектаторы -> до 1 часа&#x20;
{% endhint %}

***

### C.4 FreeKill

Намеренное нанесение урона или убийство связанного, безоружного в сопровождении человека противоположной команды и игрока нейтрального класса разрешено только в случае:

{% hint style="info" %}
* Побега / попытки побега
* Сопротивления
* Неподчинения приказам
* Цель вооружена
{% endhint %}

{% hint style="danger" %}
Предупреждение -> 1 час \* кол-во убийств&#x20;
{% endhint %}

## Events

Ниже приведены правила, которые относятся исключительно к серверу для Ивентов.

### E1. **Неадекватное поведение**

Запрещено портить суть ивента и RP другим игрокам и вести себя неадекватно в разговоре/процессе игры с другими игроками/сотрудниками (если не подразумевается отыгрышем роли/сутью ивента):

{% hint style="info" %}
* любые виды оскорблений/провокаций/угроз
* раздражающие звуки/спамить
* игнорировать просьбы других игроков/сотрудников
* Красть предметы
* Связывать игроков без причины
* Кидать гранаты без причины
* Обманывать игроков
* И т.д.
{% endhint %}

{% hint style="danger" %}
предупреждение -> бан до 3 час (зависит от нарушения)
{% endhint %}

Мы понимаем, что игроки могут развеселиться и начать делать/говорить не совсем хорошие, но смешные вещи. Однако, нужно помнить о том, что может быть весело для тебя, будет обидно для другого… так что, пожалуйста, знайте меру веселья.

### E2. **Саунпад**

Данный пункт может быть разрешен сотрудниками при проведении NRP ивентов(уточняйте у проводящего), но при RP ивентах он запрещен.

{% hint style="info" %}
Использование войс чейнджера уточняйте у проводящего, при несогласии выключить программу, можете получить мут/отстранение от ивента.
{% endhint %}

{% hint style="success" %}
При дополнении к “Городу” роли “Радио” разрешено проигрывать адекватной музыку на средней\маленькой громкости.
{% endhint %}

{% hint style="danger" %}
Предупреждение -> мут -> бан до 3 час
{% endhint %}
